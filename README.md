CoreElec Ugoos X3 DTS and DTB files to auto power USB ports
-----------------------------------------------------------

This repository compiles a Ugoos X3 DTS file that incorporates **auto power of USB ports when booting from MicroSD card**:  
cf. https://discourse.coreelec.org/t/ugoos-x3-pro-s905x3-cant-enable-usb-ports-after-booting-from-microsd-card/58818  
([archived here](https://web.archive.org/web/20260530102236/https://discourse.coreelec.org/t/ugoos-x3-pro-s905x3-cant-enable-usb-ports-after-booting-from-microsd-card/58818))

Both DTS and DTB files are published in the **[releases section](https://github.com/maaaaz/coreelec_ugoos_x3_dtb/releases)**. 
For reference, debugging or general help, I also previously dumped the [DTS and DTB from a live instance of Android 9](https://github.com/maaaaz/coreelec_ugoos_x3_dtb/tree/main/live_dump_from_Android_9) for this box.
